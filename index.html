Yan.js

Add emoticons to your input bar. ヘ(^_^ヘ)

Install

bower install yan.js

Demo

var target = document.getElementById('demo-btn');
Yan.init(target, {
  onSelect: function(emoticon) {
    document.getElementById('demo-input').value += emoticon + ' ';
  }
});
